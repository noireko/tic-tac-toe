Classic **Tic-Tac-Toe** game built using **HTML, CSS, and vanilla JavaScript**, focused on practicing programming logic, DOM manipulation, and clean code structure.

The project allows players to enter their names, alternates turns, detects the winner, and keeps a **score counter** across multiple rounds.

---

## Features

- Interactive 3x3 game board
- Player name input
- Alternating turns (X / O)
- Automatic winner detection
- Persistent score counter
- Board reset after each round
- Clear separation between game logic and UI

---

## Concepts Practiced

- Two-dimensional arrays (game board matrix)
- Separation of concerns (logic vs rendering)
- `data-attributes` to map HTML cells to board positions
- Factory Functions (`createUser`)
- Event handling (`addEventListener`)
- Array methods (`some`)
- Game state management

---

## Technologies Used

- HTML5  
- CSS3 (Grid Layout)  
- JavaScript (ES6+)

No frameworks or external libraries were used.
